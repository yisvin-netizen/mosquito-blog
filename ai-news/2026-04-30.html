🔥 AI 大事件

1. GPT-5.5 发布：OpenAI 向超级应用迈进

OpenAI 于 4 月 23 日发布 GPT-5.5（代号 Spud），登陆 ChatGPT 和 Codex 付费版。新模型在上下文处理、编码、计算机操作、网络安全和 token 效率上均有提升，同时 OpenAI 宣布 Workspace Agents 功能现已向 Business、Enterprise、Edu 和 Teachers 用户开放，逐步取代 Custom GPTs。

2. Claude Mythos 5 引发安全担忧

Anthropic 确认 Claude Mythos 5 于 4 月触发最高安全等级 ASL-4，该模型拥有 10 万亿参数规模，目前仅向部分企业和政府机构开放。4 月 21 日，Anthropic 宣布正在调查该模型的未授权访问事件，引发业界对前沿 AI 安全的新一轮讨论。

3. 谷歌 400 亿美元投资 Anthropic

谷歌计划向 Anthropic 投资至多 400 亿美元（含现金和算力支持），以应对市场对 Claude 和 AI 基础设施的强劲需求。这是继微软与 OpenAI 合作后，又一科技巨头与 AI 独角兽的深度绑定。

4. 中国叫停 Meta 收购 Manus

中国监管机构阻止了 Meta 以超 20 亿美元收购 AI 创业公司 Manus 的交易，Meta 布局自主 AI Agent 的计划受挫，中美科技投资审查趋严的信号更加明确。

5. 微软与 OpenAI 结束独家云合作

微软和 OpenAI 宣布重组合作关系，结束云服务独家协议，但 Azure 仍作为 OpenAI 主要云合作伙伴，同时删除了与通用人工智能（AGI）相关的条款。

🚀 开源与产品

DeepSeek V4 预览版发布

DeepSeek V4 于 4 月 24 日发布预览版（V4-Pro / V4-Flash），支持 100 万 token 上下文，开放权重，声称在编码和推理能力上有显著提升，但实际输出质量受到早期用户质疑。

Gemini 企业 Agent 平台上线

谷歌在 Google Cloud Next 2026 上推出 Gemini Enterprise Agent Platform，将 Vertex AI 打造为统一的企业 AI Agent 构建、治理、安全和优化平台，同时发布新版本 TPU。

Claude Connectors 扩展至 Adobe、Blender、Autodesk

Anthropic 的 Claude Connectors 进一步扩展，覆盖设计、视频、3D 建模和视觉特效工作流，AI 助手正加速渗透专业创作工具链。

GitHub 热榜：Skills 生态爆发

本周 GitHub 趋势显示，Karpathy 的 Skills 仓库再次登顶（周增 25k+ stars），Skills 相关仓库占据前五名中的三席；Nexu-io/open-design 克隆了 Claude Design（Anthropic 发布仅 12 天后）；GenericAgent 发布自演进 Agent 论文。

🧠 今日观点

"AI 竞争已从模型性能竞赛转向'分发、算力、控制'的新阶段。"

4 月最大的动作不是某款新模型的发布，而是融资轮次、重组合并、政策干预——这标志着 AI 产业正式进入制度化竞争时代。模型能力固然重要，但决定未来格局的，将是算力储备、数据资产和监管关系。
