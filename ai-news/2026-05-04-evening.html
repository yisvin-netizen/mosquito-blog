🔥 今日 AI 热点 TOP 5

1️⃣ OpenAI o1 在急诊分诊中超越人类医生

哈佛大学最新试验显示，OpenAI o1 在急诊分诊场景中正确诊断率达 67%，显著超越人类医生平均水平（50-55%）。AI 已具备实时评估患者优先级的能力，但误诊责任划分仍是落地瓶颈。

标签：🏥 医疗应用 | 🏆 技术突破 | ⚖️ 伦理争议

来源：The Guardian

2️⃣ Anthropic 发布 Managed Agents 架构：大脑与手的分离

Anthropic 工程团队披露了托管 Agent 的架构设计突破——将 Claude "大脑"与执行"手"完全解耦。Session 作为外部记忆，支持任意 rewind 和位置切片。性能提升显著：p50 TTFT 下降约 60%，p95 下降超 90%。

标签：🏗️ 架构创新 | ⚡ 性能优化 | 🔧 开发者工具

来源：Anthropic Engineering Blog

3️⃣ ruflo — Claude 多 Agent 编排平台单日增长 1,834 星

开源项目 `ruvnet/ruflo` 成为 GitHub Trending 黑马，这是企业级多 Agent 协作平台，支持 MCP 集成、RAG 向量检索、Claude Code/Codex 无缝对接。反映出市场对 Agent 编排工具的强烈需求。

标签：⭐ GitHub Trending | 🏢 企业级 | 🔗 生态集成

来源：GitHub Trending

4️⃣ Chrome 正式发布 Prompt API：浏览器原生 AI 时代来临

Chrome 正式开放 Prompt API，允许网页直接调用 Gemini Nano 本地模型。这意味着 AI 能力首次原生嵌入浏览器底层，无需调用外部 API，用户隐私保护和数据脱机处理成为可能。

标签: 🌐 浏览器 | 📱 端侧 AI | 🔓 开发平台

来源：Chrome Releases Blog

5️⃣ Box CEO：AI 不是取代工程师，而是让他们效率翻倍

Box CEO Aaron Levie 提出思想实验：当每个工程师拥有 5 倍效率时，公司不会裁员，反而会雇更多工程师。他引用生命科学公司案例——算力成本下降后，企业对工程师需求反而增加。AI 的本质是满足更多未满足的需求。

标签：💼 商业洞察 | 📈 经济学 | 🎯 就业市场

来源：X / Twitter @aaronlevie
