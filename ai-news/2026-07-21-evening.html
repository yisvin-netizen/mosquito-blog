🌟 今日 5 大 AI 热点

1. 🩺 哈佛医学院发布泛癌症基础模型 COMPASS，平均准确率超 22 种现有方法 8.5%

技术突破行业应用

哈佛医学院、罗氏制药、浙江大学联合在 *Nature Medicine* 发布 COMPASS 泛癌症基础模型，基于 33 种癌症类型的 10,184 个肿瘤样本训练，覆盖 7 种癌症、6 种免疫检查点抑制剂（ICI）的 16 个临床队列。关键数据：平均预测准确率比 22 种现有方法提升 8.5%，AUPRC 提升 15.7%，并能泛化到微调阶段未涉及的癌症类型和治疗方案。架构分三阶段——基因语言模型（GLM）把基因当 token 学习 → 分层投影器映射到 44 个可解释免疫概念 → 分类器预测 ICI 响应。真实价值是 PD-1/PD-L1、CTLA-4 等免疫检查点抑制剂只在约 30% 患者中有效，COMPASS 可辅助个体化治疗与患者分层——这是 AI for Science 真正进入临床决策的标志性进展。

🔑 关键词：COMPASS / Nature Medicine / 泛癌症基础模型 / 免疫检查点抑制剂 / AUPRC 15.7% / 基因语言模型

2. ✋ 李飞飞团队发布 T-Rex 触觉大模型：灵巧手任务 65% vs 35%，开锁 70% vs 0%

技术突破行业应用

李飞飞、Trevor Darrell、Jitendra Malik、Pieter Abbeel、Ken Goldberg、Jim Fan 等具身智能"全明星阵容"联合发布 T-Rex——首个在灵巧手任务上系统性证明"触觉不是锦上添花，而是必选项"的工作。背景：手指精密操作时相机机位根本无法布置；同硬件同任务，把触觉信号直接拼进 VLA 模型，任务成功率反而从 17% 暴跌到 6%——触觉想加就能加？视觉跑 5–10Hz，触觉需 20Hz+，两种信号塞同一低频 Transformer 会"撞车"。T-Rex 用三阶段训练（22,889 小时人类视频 → 100 小时遥操作 → 100 条演示）和 Mixture-of-Transformer-Experts 三专家架构（Latent + Action 1.41B + Tactile 0.62B）在 12 项真实任务平均拿下 65% vs 35%，开锁任务 70% vs 0%。判断：谁先解决"触觉怎么加"的架构问题，谁就可能在灵巧手赛道卡位，触觉数据可能成为整个感知-决策-执行链条中数据价值最高的一环。

🔑 关键词：T-Rex / 触觉大模型 / 灵巧手 / Mixture-of-Transformer-Experts / Tactile Expert 0.62B / 具身智能

3. 🐧 Linus Torvalds 一句话点透 AI 编程局限："创可贴式补丁正在消耗维护者"

伦理争议国际大厂

Linux 之父 Linus Torvalds 在孟买开源会议上与 DH Consulting 创始人 Dirk Hohndel 对谈，把 LLM 视为工具而非魔法。核心判断：AI 生成的"错误报告 + 幻觉内容 + 创可贴式补丁"正在消耗维护者资源；对 Linux 内核这样的复杂工程，LLM 目前无法替代长期维护者的架构判断、工程经验、代码品味——"工具、语言、AI 都可以变化，但支撑大型工程的仍是信任、经验、审查机制、对复杂系统的敬畏"。Linux 7.1 也透露了一组关键变化：NTFS 子系统出现两个并行维护团队"先竞争"；计划在 7.2 移除 1993 年的 486SX 浮点仿真代码；ISDN、ATM 等无人维护的旧代码逐步退出；仍保持每 9–10 周发版节奏。给行业的信号：当 AI 编程工具普及，最稀缺的资源不是代码生成速度，而是长期维护能力与系统级的工程品味。

🔑 关键词：Linus Torvalds / AI 编程局限 / Linux 内核 / 创可贴式补丁 / 系统工程品味

4. 💰 AI 巨头争夺人类的"错题本"：Anthropic 470 亿 ARR 反超 OpenAI，数据飞轮成新护城河

国际大厂行业应用

36 氪深度：Anthropic 用数据飞轮反超 OpenAI，ARR 从 140 亿暴涨至 470 亿，估值近万亿，企业市场份额 34.4%。核心论据：GitHub 囤的是"已经凝固的成品代码"，编辑器里流动的操作却记录着"代码形成的过程"——谁占领 coding / 办公 Agent，谁就能让用户一边付钱一边提供改进下一代模型的真实反馈。数据飞轮三要素：进入值钱的生意（coding 强对错信号 + 办公长任务链）+ 强反馈信号（代码能编译、测试能通过）+ 配套 Harness（把矛盾行为炼成训练数据）。巨头动作同步：马斯克 600 亿美元砸向 Cursor（"要的不是编辑器，是所有程序员敲键盘的声音"）；Meta 一边部署 7GW 算力（明年翻倍到 14GW），一边把新模型价格压到对手 1/4——低价不是目的，让更多 Agent 用上 Meta 模型才是目的；OpenAI 7 月 10 日新版桌面端把 Codex 并入 Chat / Work 三标签，紧急抢回程序员终端。判断：AI 的护城河已不在模型权重，而在 Harness 与数据飞轮。

🔑 关键词：Anthropic ARR 470 亿 / 数据飞轮 / Claude Code / Claude Cowork / Cursor 600 亿 / Meta 7GW

5. 🌏 HackerNews 榜首热议"中国开源权重路线正在赢"，Qwen/DeepSeek/Kimi/GLM 形成结构性优势

国际大厂技术突破

💡 一句话总结

今天的 AI 圈在四条线上同时推进：技术上 COMPASS 把泛癌症预测准确率推到 +8.5%，T-Rex 把触觉从"锦上添花"变成"必选项"，AI for Science 与具身智能同时拿到标志性进展；伦理上 Linus 一句话点透 AI 编程局限，"创可贴式补丁正在消耗维护者"成为开源圈共识；商业上 Anthropic 470 亿 ARR 反超 OpenAI，巨头争夺的是"人类犯错时被记录下来的那一刻"——数据飞轮替代模型权重成为新护城河；地缘上 HackerNews 榜首热议中国开源权重路线正在赢，Kimi Work / Kimi CLI 三层布局成型。当模型权重越来越商品化，下半年的胜负手是 Harness、数据飞轮、行业落地——谁能率先把"用户工作流 → 反馈信号 → 训练数据"这条闭环跑通，谁就拿到下一代 AI 的入场券。
