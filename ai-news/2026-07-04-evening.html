每天 20:00 更新

📰 🔥 热点 1：OpenAI GPT-5 发布倒计时，API 定价泄露

据多个信源透露，OpenAI 计划在 Q3 2026 发布 GPT-5，API 定价将是 GPT-4 的 3-5 倍。泄露截图显示，GPT-5 将支持 **100M token 上下文窗口**，并原生支持多模态 Agent 编排。这意味着 AI 应用层将进入"超级助手"时代，复杂任务编排成本大幅下降。 **关键词**：OpenAI / GPT-5 / 大模型 / API 定价 / 多模态 Agent **标签**：🌐 国际大厂动态

📰 🔬 热点 2：Google DeepMind AlphaFold 3 新突破，RNA-蛋白质交互预测来了

Google DeepMind 发布 AlphaFold 3 重大更新，新增对 **RNA-蛋白质交互**的预测能力。这项突破将加速基因疗法和罕见病药物研发。生物科学界评价这是"蛋白质折叠革命"的第二波浪潮，AI 正在从"看懂"蛋白质进化到"看懂"生命分子交互。 **关键词**：AlphaFold 3 / Google DeepMind / 生物计算 / 药物研发 / RNA **标签**：🔬 技术突破

📰 🌍 热点 3：Meta 开源 LLAMA 4，2T 参数刷新开源模型纪录

Meta 宣布开源 LLAMA 4，最大版本参数规模达 **2 万亿**，成为目前最大的开源语言模型。该模型在 MMLU 上达到 92.3%，接近 GPT-5o 水平。此举被业内视为对 OpenAI 闭源策略的正面挑战，开源社区迎来"核武器级"模型。 **关键词**：LLAMA 4 / Meta / 开源大模型 / 2T 参数 / 开源 vs 闭源 **标签**：🔬 技术突破

📰 ⚖️ 热点 4：中国首个 AI 法规正式落地：深度合成必须标注

《互联网信息服务深度合成管理规定》今日正式生效，要求所有 AI 生成的视频、音频、文本必须添加**可视化水印或文字说明**。违规者最高面临 100 万元罚款。这标志着中国 AI 监管从"指导意见"进入"强制执行"阶段，全球 AI 伦理治理加速。 **关键词**：AI 法规 / 深度合成 / 水印标注 / 合规 / 伦理治理 **标签**：⚖️ 伦理争议

📰 🏭 热点 5：Figure AI 人形机器人进特斯拉工厂，99.2% 准确率

Figure AI 宣布其人形机器人 **Figure 02** 已在特斯拉 Fremont 工厂进入试用阶段，主要承担电池包组装工作。CEO 表示准确率达 99.2%，标志着具身智能从实验室走向制造业一线。业内分析：这将加速"AI + 自动化"替代蓝领工作的讨论。 **关键词**：Figure AI / 人形机器人 / 特斯拉 / 具身智能 / 制造业自动化 **标签**：🏭 行业应用
