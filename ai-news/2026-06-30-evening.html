1Qwen 3.6 27B：开源本地模型的"甜蜜点"

阿里通义千问发布 Qwen 3.6 27B，被业界誉为本地大模型的完美选择。作为首个真正适合作为"通用智能"的本地模型，它支持 llama.cpp 运行，MacBook 即可跑满算力。通过"量子力学+舞蹈"诗歌测试和 Hexagonal Minesweeper 开发测试验证了其强大的推理能力。推荐使用 unsloth/bartowski 的 GGUF 8-bit 量化版本，体积减半但质量几乎无损。

2vLLM Micro-Agent：让一次 API 调用背后运行一个模型团队

vLLM 发布 Semantic Router 的核心突破——Micro-Agent 架构。一个 API 调用背后可以是多个模型协作，支持 5 种 Looper 算法：Confidence（低成本模型先行，置信度不够再升级）、Ratings（多候选并发+评分加权聚合）、ReMoM（多模型推理→汇总→最终综合）、Fusion（评审团模式）、Workflows（微代理工作流运行时）。用户感受仍是"一次 API 调用"，幕后复杂度被完全封装。

3Ornith-1.0：自改进的代码代理开源模型面世

专注于 agentic coding 的自改进开源模型 Ornith-1 正式发布。该模型能够基于执行反馈自主优化代码生成策略，适用于自动化编码、代码修复、多轮调试等场景。相比传统代码生成模型，Ornith-1 实现了"持续自优化"能力，大幅降低人工 review 成本。作为开源可部署方案，开发者可将其集成到本地开发环境。

据多方消息，阿里云前首席架构师、Lepton AI 创始人贾扬清在英伟达收购 Lepton AI 仅一年后已离职。贾扬清是深度学习框架领域的传奇人物，曾主导 Caffe、TensorFlow、Pytorch 的核心开发，其离职动向引发行业广泛关注。知情人士透露，他可能重新创业或加入新团队。
