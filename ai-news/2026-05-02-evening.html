🔥 今日 AI 五大热点

1. DeepMind 创始人 Demis Hassabis AGI 路线图曝光：20 年任务，进展正常

DeepMind 创始人 Demis Hassabis 在 Training Data 播客中透露 AGI 进展完全符合 2010 年预测，并宣布 AlphaFold 只是"虚拟细胞"计划的第一步，Isomorphic Labs 目标将药物研发湿实验缩短到几天。他提出大胆理论：信息可能是宇宙最基础的构建块。Demis 重申 DeepMind 使命——先解决智能（AGI），再用它解决一切。

"Our original mission statement at DeepMind was step one, solve intelligence, step two, use it to solve everything else."
"Every thing that is computable can be computed by a relatively simple description of a machine."

**关键词**：#AGI #DeepMind #AlphaFold #Hassabis #药物研发

---

2. Sam Altman 宣布 Codex 大升级：支持非编码计算机任务

OpenAI CEO Sam Altman 宣布 Codex 新版本支持非编码的计算机任务，可以用自然语言操控电脑完成复杂操作。Altman 幽默发帖称："artificial goblin intelligence — achieved"。这次升级意味着 AI Agent 从"写代码"进化到"用电脑"，是 AI 桌面应用的重要里程碑。

**关键词**：#OpenAI #Codex #Altman #AI-Agent #桌面应用

---

3. Claude Security 公开测试版上线：代码漏洞扫描进入 AI 时代

Anthropic 宣布 Claude Security 公开测试版正式上线，集成在 Claude Code Web 中，可对代码仓库进行漏洞扫描并自动修复。自 2 月研究预览以来已有数百个组织在生产代码中使用，发现了现有扫描器遗漏的高危漏洞。新增定时扫描、目录级定位、CSV/Markdown 导出等功能，安全问题终于进入 AI 驱动的新阶段。

**关键词**：#Anthropic #Claude #安全 #代码审计 #AI-Security

---

4. Andrej Karpathy 预言 LLM 三大新能力边界：menugen、.md skills、Agent Native Economy

前 Tesla AI 总监、OpenAI 创始团队成员 Andrej Karpathy 在 Sequoia Ascent 2026 炉边谈话中提出 LLM 的三大新能力：

1. **menugen**：LLM 完全吞噬的 App，无需传统代码

2. **.md skills**：用自然语言代替 Shell 脚本

3. **LLM 知识库**：对非结构化数据的计算

他预言"Agent Native Economy"将崛起，产品和服务将被分解为传感器、执行器和逻辑。

**关键词**：#LLM #Karpathy #Agent #menugen #技术前沿

---

5. Aaron Levie 预言 Agent 时代软件商业模式：席位 + API 配额将成主流

Box CEO Aaron Levie 深度分析了 Agent 时代软件商业模式演变：

1. 席位不会消失，但每个席位需要附带 API 配额供 AI Agent 调用

2. 有状态的 Agent 可能需要专属"席位"，但定价逻辑完全不同

3. 超额调用部分将采用消费模式——这将成为主流

Levie 预言：Agent 对数据的 API 调用量将远超人类用户，所有软件都必须提供 headless 访问方式。这是 SaaS 行业十年来最重大的商业模式变革信号。

**关键词**：#SaaS #Agent #商业模式 #Aaron-Levie #API经济
