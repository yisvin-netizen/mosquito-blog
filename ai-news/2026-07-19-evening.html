1. 🧪 GPT-5.6 撞开凸优化大门：LLM 首次以"第一作者"身份提交可同行评议的数学证明

技术突破国际大厂

HackerNews 当日榜首（435 票 / 270 评论）——OpenAI 的 GPT-5.6 仅凭一个 prompt，就把凸优化（CDC）领域悬而未决 30 年的公开难题给出全新证明，并在 GitHub 开源 codex-cdc-proof 推理链供复现。Reddit r/math 引发同行讨论，部分研究者称之为"LLM 在纯数学领域第一次做出可被同行评议的实质性推进"。

2. 🐦 Kimi K3 社区拐点：鹈鹕骑车基准 87 分超 GPT-5.6，中国开源首次在"创意约束"任务上领跑

国际大厂行业应用

月之暗面 Kimi K3 发布 48 小时内，开发者社区自发形成"鹈鹕骑自行车"基准（Pelican on a Bicycle）——要求模型在 100 token 内画出 ASCII 鹈鹕骑车图。K3 得分 87/100，超过 GPT-5.6 的 79 和 Claude Fable 5 的 82。

Simon Willison 当日发文《The Kimi K3 Moment》（176 票 / 160 评论）评价："中国开源模型第一次在「创意但受约束」任务上同时超过 OpenAI 和 Anthropic 主力"。值得注意的是，Kimi-cli（Kimi Code CLI 智能体）持续登上 GitHub Trending，中国阵营正在从"模型层"卷向"Agent 工具链"。

而"创意但受约束"恰好是 Agent 时代最难的能力之一——既要懂规则，又要懂审美。

关键词：Kimi K3 / 鹈鹕骑自行车 / Simon Willison / 中文开源 / Agent 工具链

技术突破行业应用
