🔥 今日 5 大 AI 热点

1️⃣ OpenAI 发布 GPT-5.5：首个"自主操作电脑"的智能体模型

标签：`#国际大厂` `#技术突破` `#智能体`

OpenAI 于 4 月 23 日发布 GPT-5.5（代号 Spud），立即登顶 Artificial Analysis 智能指数排行榜（得分 60），并在 OSWorld-Verified 基准测试（AI 独立操控操作系统、点击 UI 元素）中拿下 78.7% 的高分——这是 AI 首次在真实计算机操作任务中接近人类水平。OpenAI 总裁 Greg Brockman 表示："这为规模化 Agent 计算时代奠定了基础。" GPT-5.5 API 定价为每百万输入 Token 5 美元（是 GPT-5.4 的两倍），但由于内部推理效率提升约 40%，实际账单涨幅仅约 20%。当前已向付费 ChatGPT/Codex 用户开放，API 也即将推出。

来源：CNET / OpenAI Blog / Artificial Analysis

2️⃣ DeepSeek V4 重构发布：1.6 万亿参数 + 百万 Token 上下文完全开源

标签：`#国产突破` `#开源` `#大模型`

DeepSeek 于 4 月 24 日发布 V4 系列两款模型：1.6 万亿参数的 V4 Pro 和更高效的 V4 Flash，均支持 100 万 Token 上下文窗口。V4 采用全新的 Hybrid Attention Architecture（混合注意力架构），可在一长串提示中保持查询历史不丢失，同时显著降低了对硬件的要求——这意味着可在更便宜的硬件上部署，有望进一步拉低 AI 推理成本。V4 在 Artificial Analysis 排名第三（指数 58），仅次于 GPT-5.5 和 Grok 4.3，并在开源模型的世界知识基准中领先所有对手。更重要的是，V4 完全开源，再次向西方闭源巨头发起性价比挑战。

来源：CNET / DeepSeek 官方文档 / RenovateQR

3️⃣ xAI 发布 Grok 4.3：四智能体并行架构，定价仅为 GPT-5.5 的 60%

标签：`#国际大厂` `#多智能体` `#价格战`

xAI 于 4 月 23-24 日发布 Grok 4.3，立即跃居全球第二大模型（指数 59），以 $3/百万输入 Token 的价格比 GPT-5.5 低 40%。Grok 4.3 的核心创新在于多智能体架构：主智能体协调任务，Harper 子智能体实时抓取 X 平台数据并核查事实，Benjamin 处理纯逻辑与代码，Lucas 专注创意推理——四个子 AI 实时辩论后流式输出统一答案。这种架构使 Grok 在金融市场分析、实时文化趋势和快速事件检测上具有独特优势，加之其独特的 X 平台实时社交数据访问能力，成为目前最具差异化的旗舰模型。

来源：RenovateQR / Artificial Analysis

4️⃣ 百度文心大模型 4.5 Turbo 发布：推理速度提升 3 倍，成本降低 70%

标签：`#国产突破` `#大模型` `#性价比`

百度在 4 月下旬发布文心大模型 4.5 Turbo 版本，官方声称推理速度提升 3 倍，API 成本降低 70%，并新增多模态长文本理解能力。这是百度在 DeepSeek 以低价策略横扫市场后的一次正面反击，旨在吸引对成本敏感的企业用户。同时，百度宣布开放 128K Token 的超长上下文 API，以对标 Gemini 3.1 Pro 和 DeepSeek V4 的长文本能力。业内分析认为，国内大模型的价格战已进入白热化阶段，中小型厂商的生存空间正在被急剧压缩。

来源：36 氪 / 量子位

5️⃣ 白宫指责中国"工业级 AI 技术窃取"，中美 AI 博弈再度升温

标签：`#伦理争议` `#国际关系` `#政策风险`

4 月 23 日，白宫发表声明，指责中国以"工业级规模"窃取美国 AI 技术，并点名 DeepSeek。同日，OpenAI 也向 Bloomberg 透露，已掌握 DeepSeek 使用 OpenAI 模型进行知识蒸馏以获取竞争优势的证据。这一系列动作正值 DeepSeek V4 开源发布期间，被认为具有明显的政治意涵。中美 AI 博弈已从"技术竞争"升级为"规则制定权"的争夺——开源模型是否构成技术扩散？知识蒸馏是否违反服务条款？这些问题正在成为国际社会的新焦点。

来源：Reuters / Reuters UK / Bloomberg
