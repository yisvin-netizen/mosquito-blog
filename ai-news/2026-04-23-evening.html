🔥 今日 5 大热点

1️⃣【国际大厂动态】OpenAI 推出 ChatGPT 工作区智能体

关键词: #OpenAI #AI 智能体 #企业自动化 #工作流

OpenAI 正式发布工作区智能体功能，团队可创建共享智能体处理复杂任务和长期工作流。基于 Codex 驱动，支持在 ChatGPT 和 Slack 中共享，提供软件审查员、产品反馈路由器等 5 个预设模板。企业级监控和权限控制，5 月 6 日起向 Business/Enterprise 用户收费。这标志着 AI 从个人助手向团队工作流基础设施的重大转变。

2️⃣【技术突破】Google 发布第八代 TPU：专为智能体时代设计

关键词: #Google #TPU #AI 芯片 #硬件加速

Google 在 Cloud Next 大会发布 TPU 8t（训练）和 TPU 8i（推理）两颗芯片。TPU 8t 单 pod 扩展至 9,600 芯片，算力 121 ExaFlops，有效计算时间 97%；TPU 8i 采用 Axion ARM 架构，性能/美元提升 80%，专为 MoE 模型和智能体协作优化。下半年正式可用，显示硬件设计开始针对训练/推理分离和 AI Agent 特性进行专门优化。

3️⃣【技术突破】Qwen3.6-27B 开源：27B 稠密模型编码能力超越 397B MoE

关键词: #Qwen #开源模型 #稠密架构 #编码能力

阿里云发布 Qwen3.6-27B，在 SWE-bench Verified(77.2)、SWE-bench Pro(53.5)、Terminal-Bench 2.0(59.3) 等编码基准测试中全面超越上代旗舰 Qwen3.5-397B-A17B。稠密架构无需 MoE 路由复杂性，部署更简单，支持多模态思考和推理模式。已集成 OpenClaw、Claude Code 等编码助手，权重可在 Hugging Face 和 ModelScope 下载。证明精心训练的中小模型可超越超大规模 MoE。

4️⃣【行业应用】乒乓球机器人 ACE 首次击败顶级人类选手

关键词: #机器人 #体育科技 #实时决策 #人机对抗

名为"ACE"的乒乓球机器人在正式比赛中首次击败顶级人类选手，标志着机器人在高速动态环境中的决策能力取得重大突破。采用高速视觉系统和预测算法，反应时间低于人类极限。这项技术可应用于工业质检、自动驾驶等需要快速响应的场景，展示 AI 在物理世界交互能力的进步。

5️⃣【伦理争议】GitHub CLI 开始收集伪匿名遥测数据引发隐私担忧

关键词: #GitHub #隐私争议 #数据收集 #开源社区

GitHub CLI 官方宣布开始收集伪匿名使用数据，目前 HackerNews 讨论热度达 359 点。争议焦点在于数据收集范围不透明，部分开发者呼吁提供完全离线模式。这反映了开源社区对隐私保护的敏感性，以及工具类软件在商业化和用户信任之间的平衡难题。建议关注后续 Mozilla 和社区的回应。
