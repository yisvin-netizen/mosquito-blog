厂商动态

Astral（Ruff、uv、ty 背后团队）宣布加入 OpenAI 的 Codex 团队。这是 Python 开发生态的重大事件——Ruff 是最快 Python 代码检查工具（比 Flake8 快 100 倍），uv 是最新 Python 包管理器，两者月下载量均超数亿次。OpenAI 承诺交易完成后继续支持这些开源工具，可以预见 Codex AI 编程助手将与 Python 工具链深度整合。

2️⃣ 中国模型登顶！阿里通义千问 3.5-Max 冲击全球前五

技术突破

阿里云发布通义千问 3.5-Max-Preview，在全球知名的第三方评测平台 LM Arena 获得 1464 分的综合成绩，一跃成为全球排名第五的模型。尤其在数学能力上排名全球第五，整体胜率排名第六。这是国产大模型在国际权威榜单上的历史性突破，标志着中国大模型已进入全球第一梯队。

关键词：#阿里云#通义千问#国产大模型#LM Arena

3️⃣ 腾讯 QClaw 正式公测：微信远程操控电脑的 AI "小龙虾"

行业应用

腾讯 AI 产品 QClaw 结束内测，正式开启公测，无需邀请码即可下载使用。它让用户通过微信小程序远程控制 PC，包括文件编辑、浏览器操作、命令执行、邮件草稿等。基于 OpenClaw 开源框架，支持 Kimi 等国产大模型，内置超 5000 个技能。

关键词：#腾讯#QClaw#AI Agent#微信小程序

4️⃣ Claude Code 发布 Channels 功能：Telegram/Discord 远程控制

工具升级

Anthropic 为 Claude Code 推出 Channels 功能，允许通过 MCP 服务器将 Telegram/Discord 消息推送到运行中的 Claude Code 会话。用户可远程发送指令，Claude 执行后回复到对应平台。这是开发者远程工作流的重要升级，建立在 Anthropic 官方维护的插件系统上。

关键词：#Claude#Anthropic#MCP#Channels

5️⃣ ElevenLabs 推出 AI 音乐交易市场：首创"创作者分成"模式

伦理争议

AI 配音龙头 ElevenLabs 进军 AI 音乐市场，推出"创作者分成"商业模式——艺术家上传作品后，每次被其他创作者使用翻唱或改编，均可获得收益分成。这比传统流媒体版税更透明，且直接与 AI 创作者经济挂钩。

关键词：#ElevenLabs#AI音乐#版权#创作者经济
