## 解决方案

### 核心工具

| 工具 | 作用 |
|------|------|
| OpenClaw | AI 自动化助手 |
| wenyan-cli | Markdown 转微信公众号排版 |

### 安装配置

```bash
npm install -g @wenyan-md/cli
export WECHAT_APP_ID="你的AppID"
export WECHAT_APP_SECRET="你的密钥"
```

### 发布命令

```bash
wenyan publish -f article.md -t lapis
```

## 总结

1. 用 Markdown 写文章（专注内容）
2. 一键转换为公众号格式（自动排版）
3. 自动发布到草稿箱（一键发布）

把时间花在内容创作上，而不是重复的排版工作。
